# Thrower Gölge Blink (10)

**Tarih:** 11 Temmuz 2026

## Kullanım
1. Liste → **0 / Golge** seç
2. **X** (gamepad Square) → atıcı bölgesinde X aynasına blink
3. Eski yerde **gölge** kalır (~5 sn)
4. Sol tık atınca gölge de **aynı hedefe** çapraz ikinci top atar

Kombo: **0 Golge + 4 Sekten** → blink, iki noktadan seken çapraz.

## Notlar
- CD: 10 sn (gölge varken yeniden blink yok)
- Echo kuvveti ~%88
